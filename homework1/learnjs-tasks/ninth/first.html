Задача #1

Каким будет результат этих выражений?

Код:

// 5 больше 4 - простое численное сранение

5 > 4 // true

// "ананас" меньше "яблоко" - сравнение строк(по «лексикографическому» порядку)

"ананас" > "яблоко" // false

// "2" больше "12" - сравнение строк(по «лексикографическому» порядку)

"2" > "12" // true

// Особое сравнение null и undefined - будет true так как используется нестрогое сравнение

undefined == null // true

// Особое сравнение null и undefined - будет false так как используется строгое сравнение

undefined === null // false

// null сранивается со строкой нестрогим сравнением,

// null преобразуется к числу 0, а строка к 1

// поэтому в итогу получится false

null == "\n0\n" // false

// при строгом сравнении с null всегда возвращается false

null === +"\n0\n" // false
